# 🚆 Train Booking Application

A simple Java-based backend console based application for managing train bookings.  
This project allows users to register, log in, search for trains, book tickets, and view their bookings.


Features

- User Authentication
    - Register a new account
    - Log in and log out

- Train Management
    - View all available trains
    - Search trains by source and destination

- Ticket Booking
    - Book tickets for a selected train
    - View all booked tickets by the user
